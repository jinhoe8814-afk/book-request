초기 설정

Google Apps Script URL을 입력하면 모든 기기에서 동시에 사용할 수 있습니다.

1
Google 스프레드시트 생성 → 확장 프로그램 → Apps Script
2
Code.gs 내용 전체 붙여넣기 후 저장
3
배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포
4
생성된 웹앱 URL을 아래에 붙여넣기
developers.kakao.com에서 앱 생성 후 REST API 키를 발급받으세요.
